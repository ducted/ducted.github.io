Sending events to Elasticsearch

Outputs are plugins which receive event objects and sends them on to something else. 

outputs:
    - output: duct.outputs.elasticsearch.ElasticSearch
      server: 127.0.0.1
      port: 9200


Collecting CPU usage metrics

Sources can poll metrics from a service, or receive them directly by implementing a listening server. 

sources:
    - service: cpustats
      source: duct.sources.linux.basic.CPU
      interval: 1.0
